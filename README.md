# photo-rater

A photo rating website/Web app with user accounts. Photos are chosen at random and displayed one at a time to be rated by users. Users can rate photos anonymously, but account creation is required to comment on photos, and to upload and submit photos to be rated. Supabase as a backend DB.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/photo-rater.git
cd photo-rater
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
